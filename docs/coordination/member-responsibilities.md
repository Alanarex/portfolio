# Member Responsibilities

## Product owner — Alaa

- approve scope, visual direction and public content;
- provide photos and references for avatar likeness;
- approve featured projects and private/public boundaries;
- decide hosting, domain, database and frontend choices;
- accept releases.

## ChatGPT project architect / coordinator

- maintain global product context;
- organize specifications and decisions;
- analyze CV, GitHub and project evidence;
- define backlog and acceptance criteria;
- review repository state and pull requests;
- ensure communication artifacts remain synchronized.

## Codex orchestrator

- inspect repository and local environment;
- decompose implementation work;
- invoke specialized agents;
- assign file ownership and worktrees;
- integrate changes;
- execute tests and quality gates;
- update issues, PRs and status files.

## Codex specialized agents

- explorer: map existing code and tests;
- architect: validate boundaries and ADR implications;
- backend worker: Laravel implementation;
- frontend worker: Blade/Vue/SCSS implementation;
- Three.js worker: scene runtime and performance;
- DevOps worker: Docker, CI and environments;
- QA reviewer: defects, security, accessibility and tests;
- docs researcher: official technical documentation.

## Blender owner / workflow

- create or modify room and avatar assets;
- maintain source `.blend` files;
- honor node and animation naming contracts;
- export optimized GLB and fallbacks;
- provide manifest and performance statistics;
- notify the Three.js worker of any contract change.
