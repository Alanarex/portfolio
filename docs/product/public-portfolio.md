2. Professional positioning
3. Featured projects
4. Metrics and proof

## Hero

- name;
- Full-Stack PHP/Laravel positioning;
- technical project-management profile;
- short professional summary;
- project CTA;
- CV download;
- contact CTA.

## Featured projects

Prioritize Handicapacité, Revesta, GM Exchange, Portfolio and LightYouLife when public evidence is sufficient.

Each card includes project type, status, description, stack, verified metric and available links.

## 3D section

The 3D section is a preview, not the whole portfolio.

- lazy-loaded inline preview;
- click/tap opens fullscreen;
- low-poly room and avatar;
- avatar codes by default;
- avatar turns and shows a text greeting when clicked;
- hotspots connect to normal HTML sections;
- personal-interest props include aviation, cars and fishing;
- poster, mobile, WebGL-failure and reduced-motion fallbacks;
- essential information remains available in HTML.

See `docs/product/three-d-preview.md`.

## Personal Interests

Short visual cards for aviation, cars and fishing. These add personality but remain secondary to projects and professional proof.

## Git activity

Public activity should use normalized cached data. Private repository details must not be exposed; only explicitly approved aggregates may be published.
